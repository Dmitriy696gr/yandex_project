Задачей выполнения проекта было разработать решение, которое позволило бы персонализировать предложения постоянным клиентам, увеличив тем самым покупательскую активность. 

Для выполнения задачи были применены модели машинного обучения, а именно:
1. DecisionTreeClassifier;
2. KNN;
3. LogisticRegression;
4. SVC.
Обучив каждую модель на предоставленных данных лучшей оказалась модель линейной регрессии.
